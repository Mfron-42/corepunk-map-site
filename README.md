# Carte de Kwalat — Corepunk

Carte interactive communautaire de **Corepunk** : PNJ, quêtes, ateliers, objets,
coffres, camps de monstres et de ressources, points d'intérêt et zones.

Site statique, sans backend — hébergé via GitHub Pages.

## Mise à jour

Le contenu de ce dépôt est republié automatiquement à chaque mise à jour de la
carte. Ne modifiez pas les fichiers directement ici : toute contribution passe
par le dépôt source du projet.
